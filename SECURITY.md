# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest | :x:               |

## Reporting a Vulnerability

If you discover a security vulnerability in DAS EVENTS website, please report it responsibly.

### How to Report

- **Email**: security@dasevent.in.com
- **Subject**: [Security Vulnerability] - Brief description

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Resolution**: Depends on severity, typically 2-4 weeks

### Scope

This policy covers the DAS EVENTS website at `dasevent.in.com`, including:

- Frontend application (React/Vite)
- User data handling
- Contact forms and submissions
- Authentication mechanisms

### Out of Scope

- Third-party services (hosting, analytics)
- Social media accounts
- Physical security

## Security Best Practices

- Keep dependencies updated
- Use HTTPS for all communications
- Validate all user inputs
- Protect API keys and secrets
- Regular security audits

Thank you for helping keep DAS EVENTS secure.
